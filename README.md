## 📖 Deskripsi
Praktikum ini bertujuan untuk memahami dan mengimplementasikan **spatial indexing** serta **optimasi query spasial** menggunakan PostgreSQL dan PostGIS.

Fokus utama:
- Membandingkan performa query sebelum dan sesudah penggunaan index
- Menggunakan `EXPLAIN ANALYZE`
- Mengoptimasi query spasial
- Menganalisis hubungan antar data spasial

---

## 🗂️ Dataset
Dataset yang digunakan terdiri dari 3 jenis geometri:

| Tabel | Tipe Geometri | Deskripsi |
|------|-------------|----------|
| wilayah | Polygon | Batas wilayah administratif |
| jalan | LineString | Jaringan jalan |
| fasilitas_publik | Point | Lokasi fasilitas umum |
